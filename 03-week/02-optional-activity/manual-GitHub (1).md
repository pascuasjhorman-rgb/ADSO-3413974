manual-GitHub.md \# Manual Básico de GitHub Guía sencilla para aprender
a subir archivos a GitHub.

------------------------------------------------------------------------

# 1. ¿Qué es GitHub?

GitHub es una plataforma en internet que permite guardar proyectos y
archivos usando **Git**, un sistema de control de versiones.\
Sirve para trabajar solo o en equipo y guardar cambios en el código o
documentos.

------------------------------------------------------------------------

# 2. Crear una cuenta en GitHub

1.  Ir a https://github.com
2.  Hacer clic en **Sign up**.
3.  Ingresar:
    -   Correo electrónico
    -   Contraseña
    -   Nombre de usuario
4.  Confirmar el correo.

------------------------------------------------------------------------

# 3. Crear un repositorio

Un **repositorio** es como una carpeta donde se guardan los proyectos.

Pasos:

1.  Iniciar sesión en GitHub.
2.  Hacer clic en **New repository**.
3.  Escribir:
    -   Nombre del repositorio
    -   Descripción (opcional)
4.  Seleccionar **Public** o **Private**.
5.  Hacer clic en **Create repository**.

------------------------------------------------------------------------

# 4. Subir archivos desde la página web

1.  Entrar al repositorio creado.
2.  Hacer clic en **Add file**.
3.  Seleccionar **Upload files**.
4.  Arrastrar los archivos o seleccionarlos.
5.  Escribir un mensaje de cambio (commit).
6.  Hacer clic en **Commit changes**.

------------------------------------------------------------------------

# 5. Subir archivos usando Git (desde la computadora)

Primero instalar **Git**: https://git-scm.com

### Pasos en la terminal

Clonar el repositorio:

\`\`\`bash git clone https://github.com/usuario/repositorio.git
